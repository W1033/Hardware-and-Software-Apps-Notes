## ▲ 影棚是个大魔术箱，能出来任何你想要的效果。

反推灯光，不能通过反推的地方：
- 眼神光
- 人在背景上的影子（最多当个辅助)

反推灯光，可以通过反推的地方：
- 鼻子（不够精准）
- 脖子区域 （下巴在脖子上的影子，轮廓比鼻子的影子柔和一些），鼻子的影子比较实下巴的影子比较柔，灯离的比较近，

### 四个曲线

<img src="./readme.assets/image-20231029202720546.png" alt="image-20231029202720546" style="zoom:50%;" />

RGB(明度曲线)：实际上是个复合曲线（通道），主要控制画面的亮度和对比度。

色彩曲线：红、绿、蓝曲线，主要调整画面中的色彩。

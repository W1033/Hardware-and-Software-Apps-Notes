# Adobe Illustrator



## AI 输出格式
- Al（工程文件）
- EPS（矢量图）
- TIF（位图）
- PDF（稳定）



## AI基础界面

- 快速全屏 F／TAB
- 保存: Ctrl ＋ S ／ Ctrl ＋ Shift ＋ S
- 放缩: Ctrl ＋ `+`／ Alt ＋ 滑轮
- 恢复: Ctrl ＋ 0

- 抓手工具: 空格 ＋ 鼠标拖拽
- 撤回: Ctrl ＋ Z ／ Ctrl ＋ Shift ＋ Z
- 标尺: Ctrl ＋ R
- 参考线: Ctrl ＋ `;`
- 网格: Ctrl ＋ `"`

- 轮廓模式: Ctrl ＋ Y
- 像素模式: Ctrl ＋ Alt ＋ Y
- 叠印模式: Ctrl ＋ Shift ＋ Alt ＋ Y

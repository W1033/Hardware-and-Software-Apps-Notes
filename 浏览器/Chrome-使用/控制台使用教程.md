## Chrome 导出/导入 书签
- 导出:
    + 在计算机上打开Chrome。
    + 点击右上角的 "更多(竖的...)" 图标 。
    + 依次选择书签 书签管理器。
    + 依次点击顶部的 "更多(竖的...)" 图标 导出书签。
- 导入:
    + 在计算机上打开 Chrome。
    + 点击右上角的“更多”图标 更多。
    + 依次选择书签 然后 导入书签和设置。
    + 选择包含待导入书签的程序。
    + 点击导入。
    + 点击完成。
